### Loops in TypeScript

Tutorial
-------
Loops are structures used to repeat a block of code until a specified condition is met.

    for(let i = 0; i < 3; i++) {
        console.log("Looping...");
    }

Exercise
-------
Use a loop to print numbers from 3 to 7.

Tutorial Code
-------

    for(let i = 3; i <= 7; i++) {
        // Your code to print the number
    }

Expected Output
-------

    3
    4
    5
    6
    7

Solution
-------

    for(let i = 3; i <= 7; i++) {
        console.log(i);
    }
